WordPress Security Analysis Tool

WordMash allows you to compare two different WordPress sites - a broken (infected) one and a clean one - and then create an AI classifier to determine which files and data elements are malicious (viruses, malware, etc.) and which ones were intentional modifications.

Features

  • Projects - Configure and manage comparison projects, each with a clean and dirty WordPress installation
  • Inventory - View file counts, lines of code, database tables and rows for each project
  • Statistics - Track project progress and AI classifier accuracy
  • Admin - System administration and settings

Get Started

Head to Projects to create a new comparison project or view existing ones.
